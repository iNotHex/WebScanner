# 🌐 Web Scanner

A tool for gathering information about a website.

## Features

- Get website domain
- Get server IP address
- Detect IP version (IPv4 / IPv6)
- Check website reputation
- Show total reports
- Enumerate subdomains

## Example

```text
🟢 This Web is Clear
The Web Current Domain: gpt.com
The Web Server IP is: 104.21.3.216
The Web ipVersion: IPv4
Theres (0) Total Reports on This Web

Subdomains:
example.com
example.com
example.com
```

## Installation

```bash
git clone https://github.com/iNotHex/WebScanner.git
cd WebScanner
```

Install the required packages:

```bash
pip install -r requirements.txt
```

## Usage

```bash
python web_tool.py
```

Enter the target website when prompted.

## Requirements

- Python 3.10+
- Internet connection
